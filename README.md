# Personal Portfolio Website

[![Website Preview](assets/images/preview.png)](https://anghelmarian.github.io)

A minimalist personal portfolio website featuring:
- Animated particle background with dynamic connections
- Responsive design
- Clean, modern aesthetic
- Dark theme with subtle animations

## Technologies Used
- Jekyll static site generator
- HTML5 Canvas for particle animations
- SCSS for styling
- GitHub Pages for hosting

## Live Demo
Visit the live site: [anghelmarian.github.io](https://anghelmarian.github.io)

## Features
- Interactive particle system with depth simulation
- Smooth hover effects and transitions
- Mobile-responsive layout
- Dynamic copyright year updates
- Minimalist color scheme
